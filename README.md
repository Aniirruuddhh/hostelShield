# HotelShield - Crisis Management System

This repository contains the source code for HotelShield, a crisis management and threat detection system.

## Project Structure

The project is divided into two main components:

- **`crisis-frontend/`**: The frontend application (HTML, CSS, JavaScript) that provides dashboards for monitoring, threat detection, and responder interfaces.
- **`crisis-backend/`**: The backend server (Node.js/Express) that handles API requests, AI integrations (like audio threat detection and vision API), and database interactions (Supabase).
